So many things the Dark Web relies on...linux, tor, bitcoins...are open source. So why not a bitcoin mixer? If you're mixing your coins, don't you have a right to know exactly how the mixer functions? We can't claim this is the best mixer out there. But it has some advantages over other mixers:

  • The code is open source. Take a look at the source code yourself.
  • Low mixing fee - never more than 1% (it's randomized between 0.5% and 1%).
  • No JavaScript!
